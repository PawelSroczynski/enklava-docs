# Czym jest Enklava

Enklava nie jest projektem deweloperskim. Nie jest też komuną ani manifestem. To próba odpowiedzi na jedno pytanie: jak mieszkać, żeby nie spędzić całego życia na spłacaniu kredytu i zależności od systemów, które mogą zawieść?

## Mapa projektów

- **Jurta** — całoroczna jurta mongolska jako dom i przestrzeń warsztatowa
- **NodeZero** — system energetyczny off-grid (UPS + agregat + gazyfikator)
- **CircleUnit** — piroliza biomasy, biochar, odzysk składników z uryny
- **Lab** — eksperymenty i prototypy

## Filozofia budowania

Siedlisko Enklava to kilka hektarów w centralnej Polsce. Dom ze słomy, jurta jako przestrzeń warsztatowa, ogród permakulturowy, infrastruktura NodeZero jako kręgosłup energetyczny.
